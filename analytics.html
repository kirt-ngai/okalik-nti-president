Connect the tracker

This static GitHub Pages site cannot read server logs by itself. To start counting traffic, deploy analytics-logger.gs as a Google Apps Script web app, then paste the Web App URL into assets/analytics-config.js.

  1. Create a Google Sheet named Okalik Site Analytics.
  2. Open Extensions -> Apps Script, paste analytics-logger.gs, and deploy it as a web app.
  3. Set access to Anyone and execute as Me.
  4. Copy the Web App URL into assets/analytics-config.js.
  5. Publish the updated files. New visits will start appearing here.

This local test URL only helps this dashboard read data. Public page tracking starts only after the same URL is saved into assets/analytics-config.js.
